[[Ветки в git]]

Чтобы переименовать ветку в локальном репозитории нужно воспользоваться командой:

```bash
git branch -m old_name new_name
```

Можно не указывать второй параметр, тогда первый — будет новым именем текущей ветки. 
Работа с флагом `-m` более предпочтительна, так как явное лучше, чем неявное.

А так как ветки, это просто ссылка на коммит с именем, то никаких проблем не возникнет, если ее переименовать.
