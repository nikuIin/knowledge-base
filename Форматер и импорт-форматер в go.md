Разработчики `go` сделали встроенный форматер, что очень сильно упрощает, а точнее полностью убирает споры на счет стилизации кода:) 

```shell
go fmt my_cool_file.go
```

Эта команда форматирует ваш файл. 

Можно еще скачать `импорт-форматер`, для того, чтобы очень филигранно форматировать импорты:

```shell
go install golang.org/x/tools/cmd/goimports@latest
```

```shell
goimports -l -w my_cool_file.go
```

`-l` — выводит список файлов, которые затронет `goimports`
`-w` — записывает изменение в файлы
