## Terminal
1. Open/close terminal `cntrl-/` или `space f t`
2. Open new terminal window `cmd-n`
3. Open central terminal and open __lazygit__ `space l g`.
   `<leader>gg` в `nvim`
## Code
1. `g r` — see all references (where this code are using)
2. `s s` — офигенная вещь, бегаем по функциям очень быстро (`<leader>ss` в `nvim`)
3. `<leader> c a` — список быстрых решений проблемы
4. `cmd w` – закрыть панель (например закрыть `lazygit`)
## Multicursor
1. `g a` или `cmd + shift + l` — выбрать мультикурсором все вхождения слова
2. `g l` — выбрать следующее вхождения мультикурсором
3. `cmd + alt + [клавишы внизи/вверх/влево/вправо]` — поместить курсор на строку/позицию
## Splits 
1. `⌘ + shift + \` — сплит горизонтальный
	- `⌘ + [1,2,3,4]` — создать сплит с номером (потом можно по ним также переключаться)
2. `⌘ + k right/left/up/down` — сплиты
## Поиск по проекту
1. `S s` — офигительный поиск по тексту
2. `Space /` — `grep`

## Сдвиг строк

1. `Tab` / `Shift-Tab`
2. Выделяем, потом `shift- >` /  `shift- < ` — для одного сдвига, после сдвига выделение пропадет
3. Выделяем, потом `cmd-]` / `cmd-[` — для сдвигов без сброса выделения
