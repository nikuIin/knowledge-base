[[Nginx.canvas|Nginx]] — канва

В `nginx` есть `директивы` и `контексты`.

`Директива` — пара ключ-значение, записываются через табуляцию (`\t`, `Tab`). Конец директивы определяется `;`

```nginx
worker_processes 1;
```

`Контекст` — обособленная часть, логический блок кода, который отвечает за какую-либо часть и имеет свои `директивы`. `Директивы` зависят от определенного `контекста`. Самый внешний `контекст` — `main`, он не выделяется фигурными скобками, а просто покрывает весь конфигурационный файл. `Контексты` могут быть вложенными друг в друга.

```nginx
events {
    worker_connections 1024;
}
```
