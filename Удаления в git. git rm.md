[[Git (конфиг, алиасы). git alias]]

Чтобы удалить файл из индекса и репозиторие в гите, используйте эту команду:

```bash
git rm to_remove1 git_to_remove2 etc
```

Мы можем и пользоваться обычным `rm`, но `git rm` автоматически за нас переносит все в индекс, что экономит, логично, наше время.

Если хотим удалить каталог, то используем флаг `-r`:

```bash
git rm -r some_catalog
```

Понятное дело `git rm` работает только на отслеживаемых файлах.
