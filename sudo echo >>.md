[[Linux UNIX]]

Мы не можем использовать `sudo` к записи файл, например:

```bash
sudo echo "123" >> protected
```

Если пользователь не имеет доступа `write` к `protected`, у нас выпадет ошибка, т.к. `sudo` применяется только к `echo`.

Есть два варианта:

Передаем все в команду `bash`, `-c` флаг говорит, что всю строку запускать как команду
```bash
sudo bach -c 'echo "123" >> protected'
```

А тут используем наш тройничок `tee`
```bash
echo "123" | tee -a protected
```
