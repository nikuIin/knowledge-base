Сразу просвечу 2 момента использования `...`:

1. В параметрах функции *numbers* `...int` — позволяет передавать какое угодно количество `int` параметров. То есть `add(1,2,3,4,5)`. **Важно**: это перечисление параметров, а не массив!
2. Распаковать массив/срез в перечисление параметров

```go
func main () {
    ([]int{1,2,3}...) // 2 пункт
}


func add(numbers ...int){ // 1 пункт
    var sum = 0
    for _, number := range numbers{
        sum += number
    }
    fmt.Println("sum = ", sum)
}
