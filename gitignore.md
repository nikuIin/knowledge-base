Файл `gitignore` позволяет перечислить файлы и директории, которые следует игнорировать (не заносить в индекс). 

Важные замечания:
* `/` – это для `gitignore` корень репозитория
* `file` – это тип `file`
* `file/` – это директория `file`

Он поддерживает маски `linux`:

```gitignore 
# Скрываем все .env файлы, кроме .env-example
.*env*
!.env-example

# Скрываем ВСЕ файлы .log
*.log

# Скрываем папку some-dir/ и все, что в ней находится
some-dir/* # папки (директории) считаются как /

# Скрываем файл в корне репозитория
/my-cute-file.md

```

*Совет*: качайте для своей ОС и для проекта (стека) готовый `.gitignore`, **но всегда его изучайте**
