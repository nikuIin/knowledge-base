Разобравшись с методом, определив его проблему и спроектировав, можно приступать к кодированию.

Лучший алгоритм кодирования метода таков:

1. Определяем имя метода 
2. Пишем псевдокод и рисуем метод
3. Проанализируйте написанный псевдокод и напишите описания метода. Этот метод такой-то, он принимает се, делает с этим третье. 
4. Построчно начинаем прописывать метод, чтобы 1 строка псевдокода становилась комментарием, и из нее вылезало 1-7 строк программного кода. Если вы в псевдокоде не будите ничего обобщать, за 7 строчек не вылезите. 
5. Проверьте код. У вас должна получиться система вида:
	def example_method(A):
		"""
			этот метод принимает параметр A и делает из него переменную B
		"""
		#ФИГАЧИМ_ПЕРЕМЕННУЮ_А_В_Б
		B=A
		#ВЫВОДИМ_ПЕРЕМЕННУЮ_B
		return B
