Цикл for стоит использовать только тогда, когда программа знает, сколько итераций нужно завершить. ==__Не манипулируйте значением инкремента__==. Если условия цикла проверяется внутри дополнительно, лучше использовать цикл while. For используйте для проходке по массиву или другой структуре данных или же для тех ситуаций, когда кол-во итераций заранее известно.
