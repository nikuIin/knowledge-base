Если у вас есть например связный список и вам нужно вставить новую переменную внутрь списка, то вам нужны такие данные, как:
	node
	node.next
	node.next.next
1 переменная используется для 3-х задач, вместо этого нужно написать:
current_node = node
new_middle_node = node.next
following_node = node.next.next

Не жалейте переменные, ваша главная задача - чтобы ваш код ЧИТАЛИ, а не РАЗБИРАЛИСЬ в нем.
