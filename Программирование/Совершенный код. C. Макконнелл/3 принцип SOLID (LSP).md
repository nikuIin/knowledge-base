Метод барбариков (я так назвал)).

==Если у нас есть класс А и в нем есть метод 1, а класс B наследует класс A и переопределяет метод 1 в метод 2, то __метод 1 и 2 должны работать с одними и теми же данными и выдать ответ в одной и той же сфере__.==

Если метод 1 возвращает процентную ставку по кредиту для банка по одной формуле, то и метод 2 должен возвращать ставку для банка, но условно по другой формуле, а не возвращать ставку для клиента банка.
