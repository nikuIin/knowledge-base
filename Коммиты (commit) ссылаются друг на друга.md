[[Версионность git (коммиты)]]

Все коммиты (кроме первого) ссылаются на своего родителя. Это связь является однонаправленой, то есть потомок ссылается на своих родителей (у потомка может быть несколько родителей, у родителей может быть несколько потомков)

Тем самым коммиты образуют связь между собой.
