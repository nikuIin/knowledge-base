[[Git (конфиг, алиасы). git alias]]

`git mv` очень похож по логике на `git rm` ([[Удаления в git. git rm]]).
Он также работает только на файлах/директория, которые отслеживаются гитом.

Работает как и дефолтных `mv`, но в добавок сразу перемещает все в индексе.

```bash
git mv some some.txt
```

В `git` такие файлы помечаются как *renamed *.
