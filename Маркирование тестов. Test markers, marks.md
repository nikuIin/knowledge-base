[[Тестирование. Pytest]]

Тесты можно группировать не только с помощью классов. Им можно определить общие теги.

Задаются они декоратором:
`@mark.my_cool_mark`

`pydantic` будет жаловаться на кастомные маркеры, поэтому их вам нужно зарегистрировать. 

Это делается в файле конфигурации. Добавляем в `pytest.ini` переменную `markers` и описываем там маркеры:

```ini
[pytest]
pythonpath = . src

markers =
    access_token: mark a test, that relates using acces_token
    refresh_token: mark a test, that relates usingrefresh_token
```

- Запустить тесты с определенной меткой:

```bash
pytest -m my_cool_mark
```
