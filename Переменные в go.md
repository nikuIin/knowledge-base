Переменные в `go` можно объявлять по-разному. Но так как это строго-типизированный язык, мы должны всегда (почти) объявлять тип переменной.
## Простое объявление переменной 

```go
var my_cool_var string
```

Если хотим объявить несколько

```go
var my_cool_var, my_next_cool_var string
```

## Устанавливаем значение при объявлении

```go
var my_cool_var string = "My cool var"
```

Если хотим сделать это для несколько переменных:

```go
var (
	name string = "Tom"
	age int = 12
)
```

Также мы можем назначить переменную без типа, `go` подтянет тип из значения:
```go
var name = "Tom"
```

## Краткое объявление переменной (автоматическое определение типа)

```go
func main() {
	cool_var := 12
}
```

`cool_var` будет иметь тип, который лучше всего соответствует правой части.
Такое объявление работает только в методах. Глобально так сделать не получится.
