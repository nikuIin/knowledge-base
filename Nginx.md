`NGINX` — это веб сервер, который занимается такими вещами, как:
- **обратное проксирование запросов. (reverse proxy)**  Клиенты (брауезры) все запросы шлют именно на `nginx`, это важно понимать (например для того, чтобы понять, что установка `CORS`-ов на `nginx` is good idea). А вот уже потом `nginx` проксирует эти запросы дальше, например на статитечские файлы `frontend` Либо на бэкэнд сервер, то есть с `api`.
  (`reversy proxy` и `proxy` [[reverse proxy и proxy (проксирование и обратно проксирование)]])
- **балансировка нагрузки** — `nginx` отличное решение для балансировки нагрузки
- **encrypting/decrypting** — шифрование/дешифрование данных. Именно слой `nginx` отлично подходит для внедрения `https`. 


Установка:

```bash
brew install nginx
```

Запуск:

```bash
nginx
```

Расположение конфига:

```bash
/opt/homebrew/etc/nginx
```
