Конфигурация и запуск
---

* __С верными параметрами:__
	Атрибуты переданы верно: 
		указаны и содержат пробелы и кириллические символы (повторить для форматов путей в Windows и nix файловых системах, обратить внимание на имена логических дисков и разделители имён каталогов (“/” и “\”)).
			1. DulatesOnly (not necessary argument)
			2. HtmlOutputFileName
			3. CsvOutputFileName
			4. StartingDirectoryN
* __Без параметров__.
* __С недостаточным количеством параметров.__
* __С неверными параметрами:__
	1. DulatesOnly (not necessary argument):
		 * передан неправильно или не передан
	2. HtmlOutputFileName:
		* передан неправильно или не передан
	3. CsvOutputFileName:
		* передан неправильно или не передан
	4. StartingDirectoryN:
		* передан неправильно или не передан
		* директория N равна одной из n-1..(n-b)

Поиск и создание списка файлов
---
* __Проверка доступа директории на чтение и на исполнение 
* __Работа с файлами из форматов [mp3, flac, wav, ogg, wma]__:
	1. Найти файл
	2. Проверить доступ на чтение 
	3. Проверить, если ли файл в составленном списке через md5sum или cmp 
	4. Если файл есть, первый из идентичных использовать в параметре для создание символьной ссылки, второй удалить и переименовать в формат [имя_файла]-N, где N - номер ссылки.
	5. Если файла нету, занести в список файлов
	6. При проверке последнего файла сконвертировать список в HTML и CSV форматы
	==+ разобрать исход, если файл открыт и заблокирован.==

	Закрытием окна консоли.
