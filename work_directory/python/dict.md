Словари в питоне мега классные. Работают как пары [ключ:значение]

## Получение элементов

### Через \[key]

**Вернет значение по ключу или ошибку**
```python
some_dict = {"a" : 1, ...}
some_dict["a"] # -> 1

some_dict["b"] # -> **KeyError**
```

## get()

**Вернет значению по ключу, None или дефолтное значение
```python
some_dict = {"a" : 1, ...}
some_dict.get("a") # -> 1
some_dict.get("b") # -> None
some_dict.get("b", "Ничего не нашлось") # -> Ничего не нашлось
 ```
## pop()

**Вернет и удалит пару по ключу, или вернет дефолтное значение, или вызовет ошибку**
```python
some_dict = {"a" : 1, ...}
some_dict.pop("a") # -> 1 и удалить пару "a" : 1
some_dict.pop("b") # -> **KeyError**
some_dict.pop("b", "Ничего не нашлось") # -> Ничего не нашлось
 ```
## popitem()

**Вернет и удалит последнюю добавленную пару или вернет KeyError (если словарь пуст)**
```python
a = {"a" : 123, "b": 345}
a["c"] = 678


print(a.popitem()) # ('c', 678)

print(a) # {'a': 123, 'b': 345}
 ```

## Включения

```python
a = {x:x**2 for x in range(10)}
print(a) # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25, 6: 36, 7: 49, 8: 64, 9: 81}
```

## Ограничения
**Ключем словаря должен быть обязательно немутабельный тип данных**

## Сравнения 
**Поддерживаются только `==` и `!=`**
