
## Итераторы, Итерируемые Объекты и Генераторы в Python

Итерируемые Объекты (Iterables)

•  Определение: Итерируемый объект - это любой объект в Python, который может возвращать свои элементы по одному. Другими словами, это объект, который можно использовать в цикле for.
•  Примеры:
  •  list: [1, 2, 3]
  •  tuple: (1, 2, 3)
  •  string: "Hello"
  •  set: {1, 2, 3}
  •  dict: {"a": 1, "b": 2} (итерируется по ключам)
  •  Файлы (при чтении построчно)
•  Как узнать, является ли объект итерируемым? Объект является итерируемым, если он реализует метод __iter__(), который возвращает итератор.

Итераторы (Iterators)

•  Определение: Итератор - это объект, который позволяет последовательно перебирать элементы итерируемого объекта. Он "помнит" текущее состояние обхода.
•  Ключевые особенности:
  •  Итератор получается из итерируемого объекта с помощью функции iter(iterable).
  •  Итератор реализует метод __next__(), который возвращает следующий элемент последовательности. При достижении конца последовательности, __next__() поднимает исключение StopIteration.
•  Как это работает: Когда вы пишете цикл for item in my_iterable:, Python за кулисами делает следующее:
  1. Вызывает iter(my_iterable) для получения итератора.
  2. На каждой итерации цикла вызывает next(iterator) для получения следующего элемента.
  3. Когда next(iterator) поднимает StopIteration, цикл for завершается.

Создание Итераторов вручную
```python
class MyIterator:
    def __init__(self, data):
        self.data = data
        self.index = 0

    def __iter__(self):
        return self  # Итератор возвращает сам себя

    def __next__(self):
        if self.index >= len(self.data):
            raise StopIteration
        value = self.data[self.index]
        self.index += 1
        return value

# Пример использования
my_list = [10, 20, 30]
my_iterator = MyIterator(my_list)

for item in my_iterator:
    print(item) # Output: 10, 20, 30
```

Пример создание итерируемого объекта:
```python
class Car:
    def __init__(self, *args):
        self.__compontents = args

    def __iter__(self):
        return iter(self.__compontents)


car = Car(1, 2, 3, "fdsf")

for item in car:
    print(item)
```
