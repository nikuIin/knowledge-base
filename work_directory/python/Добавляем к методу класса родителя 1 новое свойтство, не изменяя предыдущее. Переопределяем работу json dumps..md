У json.dumps() нету метода перевода в friendly формат даты datetime.datetime. Поэтому мы можем немного поиграться с классом преобразователем

```python
import json
import datetime
import time

class JSDate(json.JSONEncoder): # создаем класс наследователь от сдандартного класса преобразователя
	def default(self, obj):
		if isinstance(obj, datetime.datetime):
			return time.mkdime(obj.timetuple())
		return json.JSONEncoder.default(self, obj) # в случае если объект не дата, вызываем метод объекта родителя

```
