Сложны циклы лучше писать изнутри. 
Представьте, что вам нужно вычислить суммарный балл школы на ЕГЭ. Для этого вам нужно пройтись по всем ученикам и предметам. Начинайте с малого.

	ege_scores = []
	ege_scores = student_ege_scores[student_id]              [type_of_ege]
	total_scores += ege_scores

Потом добавьте итераторы в массив ege_scores

	ege_scores = [student_id][type_of_ege]
	ege_scores = student_ege_scores[student_id]              [type_of_ege]
	total_score += ege_scores

Теперь сделайте проход по внутреннему циклу.

	for type_of_ege in ege_list:
		ege_scores = student_ege_scores[student_id]               [type_of_ege]
		total_scores += ege_scores

Теперь оберните первый цикл:

	total_scores = 0
	for student in student_list:
		for ege in ege_list:
			ege_scores = student_ege_scores[student.id].[ege.type]
			total_scores += ege_scores
