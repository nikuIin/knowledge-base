Как вы запишите переменную id? Сделаете глобальную переменную global_id и будите добавлять в каждом соответствующем методе id = global_id++? Ну, ==это в корне разрушит всю программу.== 

Правильнее определить метод setId() и уже передавать в каждом методе id = setId(). Если вы решите изменить способ проставления ID, например зарезервируете часть их них, то вам ==не нужно будет менять десятки методов, а всего лишь 1.== А также важно ==задать id как класс==. Почему? Потому что в случае, если вам нужно будет изменить id с int на double, вы поменяете его лишь в классе, а не опять же, в десятках методах.
