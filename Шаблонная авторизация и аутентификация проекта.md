[[Шаблонный проект FastAPI]]

Для того, чтобы создать `JWT` методы аутентификации и авторизации, мы создадим отдельный класс `auth_master`, который будет отвечать за все взаимодействия с `JWT`.

Но перед этим нам нужно определить:
- модели `JWT`
- модель `user`
- репозиторий `user`
- репозиторий `JWT`

После создания всех нужных элементов, мы можем приступить к созданию самого `auth_master`. Он будет отвечать за работу с паролем:
- захешировать паоль
- проверить валидность пароля с хешом из `БД`
- сгенерировать `jwt payloads`
- декодирование токенов 
- кодирование токенов
- генерация токенов
- установка токенов
- валидация токенов
- проверка авторизации
- установка токенов в `cookies`
- очистка `cookies`
- получение токенов из `cookies`
