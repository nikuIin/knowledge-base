[[Версионность git (коммиты)]]

Что такое отмена? На самом деле отмена – это проведение обратных действий. Отмена надписи в файле «Новая строка» – это буквально ее удаление. 

Так вот, когда вам не подходит `git reset` ([[git reset, удаляем коммиты]]), так как он не сохранит информацию о удаление, а просто снесет ненужные коммиты, то можно использовать `revert`. 

`Git revert` как раз делает новый коммит с буквально противоположными действиями выбранного коммита, то есть он отменит выбранный коммит, создав анти-коммит (коммит отмены). Удобно, когда вам нужно отменить изменения быстро и сохранить это в истории.

```bash
git revert HEAD^2~3
```
